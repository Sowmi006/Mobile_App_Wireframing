# 🍽️ Restaurant Mobile App UI/UX Design (Figma Project)

## 📌 Project Overview
This project is a **UI/UX design prototype for a Restaurant Mobile Application** created using Figma.  
It focuses on providing a smooth and user-friendly food ordering experience with a clean interface and simple navigation flow.

The design includes all essential screens required for a complete food ordering workflow from splash screen to checkout.

---

## 🎯 Objective
To design an intuitive mobile app interface that allows users to:
- Browse restaurants and food items
- View restaurant details
- Add items to cart
- Proceed to checkout smoothly

---

## 🧩 Screens Included
- 🚀 Splash Screen  
- 🔐 Login / Sign Up Screen  
- 🏠 Home Screen (Restaurant Listing / Menu)  
- 🍔 Restaurant Details Screen  
- 🛒 Cart Screen  
- 💳 Checkout Screen Flow  

---

## 🛠️ Tools Used
- Figma (UI/UX Design)
- Wireframing Tools
- PNG / PDF Export for documentation

---

## 📱 Features of UI Design
- Clean and modern UI layout  
- Easy navigation between screens  
- Consistent color theme and typography  
- Cart system with order summary  
- Smooth checkout flow  
- Mobile-first responsive design approach
